扇贝阅读翻译助手

兼容OpenAI格式的API地址
deepseek-chat 通用对话
deepseek-reasoner 深度推理
自定义 手动输入
页面中侧边栏元素的CSS选择器
使用浏览器内置语音引擎，免费无需配置。音色取决于系统语音包。
使用方法
1. 打开扇贝阅读任意文章
2. 鼠标悬浮在段落上，点击"译"按钮
3. 翻译结果显示在侧边栏或右侧面板
4. 在"显示设置"中可调整显示方式
百度翻译接入
1. 访问 百度翻译开放平台
2. 注册并登录百度智能云账号
3. 开通"机器翻译"服务
4. 在"应用管理"中创建应用
5. 将API Key和Secret Key填入配置
语音朗读
1. 翻译结果旁将显示🔊朗读按钮
2. 支持朗读原文（英文）和译文（中文）
3. 使用浏览器内置语音引擎，免费无需配置
4. 在"显示设置"中可选择不同音色
DeepSeek接入
1. 访问 DeepSeek开放平台
2. 注册并登录账号
3. 进入"API Keys"页面创建Key
4. 将API Key填入配置
引擎区别
百度翻译：专业翻译API，速度快、稳定性好
DeepSeek：AI大模型，理解上下文语境更准确
